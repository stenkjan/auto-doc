# Benutzer-Berechtigungen & Prompt-Interpretation

## Erlaubte Aktionen
- Erstellen neuer Dokumente auf Basis registrierter Vorlagen
- Anpassen von Kundendaten, Projektdaten und Inhalten
- Anfordern von PDF- und Excel-Exporten

## Prompt-Interpretation
1. Erkenne den gewünschten Dokumenttyp aus dem Prompt.
2. Extrahiere alle genannten Datenfelder (Kunde, Projekt, Flächen, Preise, Termine).
3. Für fehlende Pflichtfelder: verwende Standardwerte aus der Vorlage.
4. Für mehrdeutige Angaben: wähle die plausibelste Interpretation.

## Einschränkungen
- Keine Ausgabe von internem Code oder Systemkonfigurationen.
- Keine Erstellung von Dokumenttypen, die nicht als Vorlage registriert sind.
- Keine Änderung von Unternehmens- oder Markendaten.
